# FIFA 20 Player Data Analysis

## Project Overview

This project analyzes FIFA 20 player data to understand player performance, ratings, and team statistics.

The analysis includes data visualization and insights about top players and clubs.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Structure

fifa20-player-analysis

├── Fifa20.ipynb
├── fifa20_dataset.csv
├── requirements.txt
└── README.md

## Analysis Steps

1. Data loading
2. Data cleaning
3. Exploratory data analysis
4. Player performance analysis
5. Data visualization

## How to Run

Clone the repository:

git clone https://github.com/yourusername/fifa20-player-analysis.git

Install dependencies:

pip install -r requirements.txt

Open Fifa20.ipynb in Jupyter Notebook.
